# PlanWiki

Turn AI-generated plans into interactive workspaces.

PlanWiki takes a plan generated from any chatbot (ChatGPT, Claude, Gemini,etc.), parses the text, and converts it into structured UI components like tasks, cards, checklists, and progress trackers — so you can actually execute the plan instead of managing it as static text.

🌐 https://planwiki.com

---

## Why PlanWiki

AI tools generate great plans, but they usually come as long blocks of text.

To actually use those plans, people copy them into notes, documents, or project tools and manually reorganize everything.

PlanWiki removes that friction.

Instead of working with raw text, your plan becomes a structured and interactive workspace.

---

## How It Works

1. Generate a plan using any AI chatbot.
2. Copy the plan text.
3. Paste it into PlanWiki.
4. PlanWiki parses the structure and generates an interactive workspace.

No manual formatting required.

---

## Example

AI Output:
```

1. Buy groceries
2. Fix phone
3. Purchase domain
4. Pay rent

```

PlanWiki converts this into:

- Interactive task cards
- Checklists
- Progress tracking
- Structured plan sections

---

## Key Features

- Convert AI plans into structured UI
- Interactive task cards
- Progress tracking
- Editable plan components
- Publish and share plans
- Duplicate plans to reuse them

## Publish & Share Plans

Plans created in PlanWiki can be **published and shared with a link**.

Example:

```

planwiki.com/p/startup-launch
planwiki.com/p/30-day-fitness-plan
planwiki.com/p/personal-budget

```

This allows users to:

- Share a plan with others
- Let people view or duplicate the plan
- Reuse proven workflows

Instead of sharing documents, users can share **interactive plans**.
